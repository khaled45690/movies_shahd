## Key Design Decisions

1. **Separation of Concerns**: Each component in its own file
2. **Composable Architecture**: Everything is a composable function
3. **State Management**: Centralized in MovieBoxApp
4. **Reusability**: Components accept props for flexibility
5. **Dark Theme**: Consistent branding with custom colors
6. **Grid Layout**: 2-column responsive grid for movies
7. **Empty States**: Proper UX for empty data scenarios
